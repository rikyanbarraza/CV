Web development on a Chrome OS



You can do basic web development in chromebooks togheter with the built in linux!
A cheap and great way if you don't have the economy or for emergencies.
The speed in the model C523 I have is more then fair to get in to web development.
To keep in the back of the head though, you will need some basic understanding of linux.
other option is to follow this guide blindley.
The upside is that ChromeOS will partition linux on it's on space so nothing
can get ruined in ChromeOS if you mess something up in Linux!
So if you have time I would give it a go!
